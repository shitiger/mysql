### Mysql 先排序后获取数据
想要获取时间最新的那一条数据
#### 5.6
```
SELECT
	temp.*
FROM
	(
		SELECT
			manager_id,
			manager_type,
			create_time
		FROM
			table_a
		WHERE
			user_id = 5
		AND operation = 3
		ORDER BY
			create_time DESC
	) temp
GROUP BY
	temp.user_id,
	temp.manager_type
```
